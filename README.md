House Liber Arce
---------------
AI-powered narrative operations platform for large roleplay communities.

Tech Stack
- Python, FastAPI
- Next.js (React, TypeScript)
- PostgreSQL
- Pinecone
- OpenAI
- Docker

Features
- Role-based lore access
- AI-powered lore search (RAG)
- Text roleplay threads
- Mission planning and summaries

Local Development
- docker compose up
- uvicorn ...
- npm run dev
